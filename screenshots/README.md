# Ekran Görüntüleri

Bu dizin README.md'de kullanılacak ekran görüntülerini içerir.

## Eklenmesi Gereken Ekran Görüntüleri

| Dosya Adı | Açıklama |
|-----------|----------|
| `popup.png` | Popup ayarlar arayüzünün görünümü |
| `subtitle-overlay.png` | Video üzerinde çift altyazı gösterimi (EN + TR) |
| `translation-progress.png` | Çeviri sırasında durum göstergesi |
| `cache-loaded.png` | Önbellekten yüklenen çevirinin gösterimi |

## Ekran Görüntüsü Rehberi

- **Format:** PNG
- **Genişlik:** 800–1200px arası önerilir
- **İçerik:** Kişisel bilgi (API key, e-posta vb.) görünmemeli
- **Adlandırma:** Küçük harf, tire ile ayrılmış (`popup-settings.png`)
